# 2D Platformer — Project

## Vision

A 2D side-scrolling platformer where the player switches between four distinct characters on the fly, each with unique stats and abilities required to overcome different level obstacles.

## Goals

- [ ] Deliver a playable single-level demo showcasing all four character archetypes
- [ ] Design the level so that each character's unique ability is meaningfully required
- [ ] Add sprite-based visuals using the downloaded asset pack (pipo-nekonin series)

## Tech Stack

- **Language:** Python 3.x
- **Framework:** Pygame 2.6.1
- **Assets:** pipo-nekonin sprite sheets (`sprites/character/`), tileset pack (`sprites/tileset/`)
- **Entry point:** `platformer.py`

## Target

A polished prototype / portfolio piece. Single-level, no save system, no multiplayer.
